### 1. 주제
-----

    고객 및 시장데이터를 활용한 대고객 금융 서비스 제안
  
### 2. 데이터 설명
-----

    고객데이터   

    -고객의 거래이력 및 인구통계학적 정보와 4개월간의 자산/거래/채널접속 내역  
    
    -범주화, 상하위 1% 데이터 대체, 결측치 처리 되었음   
    
    -시계열 정보(2022.12 ~ 2023.03)   
    
    -자산 평가액, 매수체결횟수   
    
    시장데이터   

    -4개월간의 일자 별 미국지수/국내지수와 이전 3년간 미래에셋증권에서 진행된 공모주 청약일정 제공   
    
    -mkt_idx, apy_itm_hist 두 개의 파일로 나뉘어 있음
